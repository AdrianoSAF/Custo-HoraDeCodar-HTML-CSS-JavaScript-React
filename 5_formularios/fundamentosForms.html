O que é um formulário? *Recebe dados e valida *Tags mais utilizadas: forms, label e input; *Tag forms: Cria o formulário e o delimita; *Tag Label: Descreve os inputs; *Tag input: inserimos dados, ele tem vários tipos. -number, email, text e etx. -Esses tipos são definidos no atributo type. Atributos da tag form *Action: o arquivo/página que os arquivos serão enviados; *method: GET ou POST: recebe ou envia dados; Atributo name *É utilizados para configurar nossos inputs; *O valor é ligado ao propósito do input. Ex: input de idade tem o atributo name como age ou idade *Esse atributo é utilizado no back-end para pegar o valor do campo. Atributo label *for: Utilizado para lincar com o input; *O valor deve ser o mesmo que o atributo name que corresponde aquele label; *Utilize o for para propósito semântico, isso ajuda o site a melhorar no rankeador do google. Enviando dados de formulário *É enviado atravez de um botão chamado submit; *Esse botão também é um input, com o type de valor submit; *Os dados serão enviados ao servidor através de uma requisição HTTP; Elemento select *Select: temas opções representadas por tags de opton; *A tag Select também tem um atributo name.
